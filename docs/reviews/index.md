# 审查记录

## 1. 文档目的

本文档作为历史审查、复盘和阶段性优化计划的统一入口。

本文件回答的是：

- 历史审查记录存放在哪里
- 哪些文档属于复盘记录而非长期规范
- 应该按什么顺序查看过去的审查结论

## 2. 使用原则

- 本目录下的文档用于保留阶段性结论、问题分解和复盘上下文。
- 本目录下的文档不是项目长期规范。
- 审查记录中的规则，只有在回写到上层主文档后，才视为正式规则。
